# Patch Tuesday Calculator

A simple web app to find the second Tuesday of any month (Patch Tuesday).

## Features

- View the next 12 Patch Tuesdays at a glance
- Look up Patch Tuesday for any month and year
- Clean, responsive design
- Zero dependencies - just HTML, CSS, and JavaScript

## Usage

Simply open `index.html` in your browser. No build step required!

## Files

- `index.html` - Main HTML structure
- `styles.css` - All styling
- `app.js` - JavaScript logic

## What is Patch Tuesday?

Patch Tuesday is the second Tuesday of each month, when Microsoft releases security patches for Windows and other software.
